# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-30

### Added
- Initial project template release
- API service skeleton with Go-Zero framework
- Model layer with dual ORM support (GORM + SQLx)
- pkg packages: response, errorx, db, validator
- Optional service components: RPC, Job, Consumer
- Interactive init-project.sh script
- Complete IDRM specification documents
- CLAUDE.md for AI-assisted development
- Makefile with common commands
