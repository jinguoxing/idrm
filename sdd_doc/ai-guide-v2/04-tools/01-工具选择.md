# 工具选择指南

> 如何选择合适的AI工具？

---

## 🤔 决策树

### 按复杂度

```
简单（<100行） → Cursor
中等（100-500行） → Cursor + (Kiro或Claude)
复杂（>500行） → Kiro + Cursor + Claude
```

### 按场景

| 场景 | 工具 |
|------|------|
| 新增功能 | Kiro.dev |
| 添加API | Cursor |
| 代码重构 | Cursor |
| 批量测试 | Claude CLI |
| Code Review | Claude CLI |

---

## 📊 工具对比

### Kiro.dev
- ⭐⭐⭐ 结构化specs
- ⭐⭐⭐ 任务追踪
- 适合：大型功能、团队协作

### Cursor
- ⭐⭐⭐ 开发速度快
- ⭐⭐⭐ IDE集成
- 适合：日常开发、快速迭代

### Claude CLI
- ⭐⭐⭐ 批量处理
- ⭐⭐⭐ CI/CD集成
- 适合：自动化、批量操作

Kiro + Cursor + Claude
